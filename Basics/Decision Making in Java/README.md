# Decision Making in Java

## Problem Statement:
Given two integers, `n` and `m`, check their relation and return:
- `"lesser"` if `n < m`
- `"equal"` if `n == m`
- `"greater"` if `n > m`

---

## Examples:

### Example 1:
**Input:**  
`n = 4, m = 8`  
**Output:**  
`lesser`  
**Explanation:**  
4 < 8, so the output is `"lesser"`.

### Example 2:
**Input:**  
`n = 8, m = 8`  
**Output:**  
`equal`  
**Explanation:**  
8 == 8, so the output is `"equal"`.

---

## Instructions:
Write a program in Java to solve the problem.
